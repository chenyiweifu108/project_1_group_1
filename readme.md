# Project Reproducibility Instructions

To reproduce the results presented in this project, please follow the steps below.

1. Clone or download this repository to your local machine.

2. Make sure the following dataset files are located in the same directory as the notebook:

WHR_2023.csv  
newcountryvars.csv

3. Install the required Python libraries using the provided requirements file:

pip install -r requirements.txt

4. Open the notebook:

Project_1_Group_3.ipynb

5. Run all cells sequentially from top to bottom.

The notebook contains all steps necessary to reproduce the results, including data preprocessing, feature engineering, model training, model evaluation, and SHAP analysis.
